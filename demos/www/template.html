A collection of Pivot demos. All demos require Java 6 or greater.

  • "Kitchen Sink"

    Demonstrates a number of commonly used Pivot components.

    View

  • Stock Tracker

    A simple but practical "real world" application built using Pivot. Monitors stock quotes provided by Yahoo! Finance.

    View

  • iTunes Search

    Simple application that allows a user to run search queries against the iTunes Music Store and presents the results in a table view.

    View

  • Charting

    Demonstrates charting in Pivot using the JFreeChart chart provider (hosted at http://ixnay.biz).

    View

  • Decorator

    Decorators allow developer attach additional presentation to components, such as drop shadows, reflections, image effects, etc.

    View

Other Demos

  • News Ticker/Slide Show Demonstrates Pivot used in a real-world context, used to show a news feed and a slide show of client logos (hosted at http://www.satelliteconsulting.com).
    View
  • JSON Viewer Allows users to visually browse a JSON structure using a TreeView component.
    View
  • Scripting Simple example of a Pivot application written using JavaScript.
