Pivot Starter Application

This starter kit includes two simple Pivot applications, one written using Java and the other using JavaScript. Both applications are otherwise identical. Instructions for running each are below.

Java

Requires Java 5 or greater to run and Ant 1.7 to build.

  • To run as an applet, view starter.html.
  • To run as a desktop application, execute run.sh (UNIX) or run.bat (Windows) from a command prompt in the current directory (UNIX users may need to assign execute permissions to the launch script).
  • To build, run ant from this directory.

JavaScript

Requires Java 6 or greater.

  • To run as an applet, view starter-script.html.
  • To run as a desktop application, execute run-script.sh (UNIX) or run-script.bat (Windows) from a command prompt in the current directory (UNIX users may need to assign execute permissions to the launch script).
